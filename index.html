🎓 Optimizing Recruitment with Machine Learning

This research paper, to be presented at IEEE AIRC 2025, explores how machine learning (ML) can enhance recruitment processes by predicting the best-fit candidates.

🔍 Research Approach:

  • 🤖 ML Models: Using models like Random Forest and CatBoost to predict top candidates.
  • 🔧 Feature Engineering: Creating relevant features to improve model accuracy.
  • ⚙️ Hyperparameter Tuning: Fine-tuning models using grid and random search methods.
  • 🔍 Model Validation: Cross-validation to ensure model reliability.

✨ Expected Results:

  • 📈 High Prediction Accuracy: Aim for 96%+ accuracy in candidate matching.
  • ⏱️ Faster Hiring: Streamline the recruitment process.
  • 💰 Cost Efficiency: Reduce recruitment costs and re-interviews.

Why it matters: This research aims to modernize recruitment, offering organizations a data-driven approach to better, faster hiring decisions.

🤖 Revolutionizing Recruitment: Bias Mitigation & Efficiency

Selected for IBAC 2025 Conference, this project focuses on optimizing recruitment by applying machine learning models and fairness-aware algorithms to reduce biases.

🔍 Research Approach:

  • 🧠 Applied models like Random Forest, SVM, and CatBoost to predict hiring decisions.
  • ⚖️ Integrated fairness metrics to reduce biases and promote equitable hiring.
  • 💡 Used SHAP and LIME for explainable AI.

✨ Results:

  • 🏆 Achieved 95% accuracy with CatBoost.
  • 💬 Reduced biases through fairness metrics.
  • 🔑 Identified key features like Education Level and Personality Score.

💡 Why it matters: This project ensures data-driven, transparent, and inclusive hiring decisions.

📦 Enhancing Operational Efficiency: Inventory Management

This project uses machine learning and time series forecasting to optimize inventory management, reducing costs and improving customer satisfaction.

🔍 Approach:

  • ✅ Applied SARIMA, Gradient Boosting, and XGBoost for forecasting.
  • ✅ Used hyperparameter tuning for optimization.

✨ Results:

  • 🚀 Identified SARIMA as the best model for accuracy.
  • 🚀 Optimized inventory by integrating Economic Order Quantity.

Why it matters: This project demonstrates how machine learning can improve inventory management and supply chain efficiency.
